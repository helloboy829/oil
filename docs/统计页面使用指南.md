# 统计页面使用指南

## 📊 功能概述

统计页面提供了强大的数据分析功能，支持多维度筛选，帮助您快速了解业务情况。

## 🎯 核心功能

### 1. 基础统计

**销售趋势图**
- 展示指定时间段内的销售额变化
- 支持按日/周/月切换统计粒度
- 折线图 + 面积填充，趋势一目了然

**商品销量排行**
- TOP10 最畅销商品
- 显示销售数量和排名

**客户消费排行**
- TOP10 消费最多的客户
- 显示消费金额和订单数

### 2. 智能筛选 🆕

#### 2.1 客户筛选
**用途**：查看某个客户的消费情况

**操作**：
1. 点击"全部客户"下拉框
2. 输入客户姓名搜索（支持模糊搜索）
3. 选择目标客户
4. 数据自动刷新

**效果**：
- 趋势图显示该客户的消费趋势
- 商品排行显示该客户购买的商品排行
- 所有统计数据仅统计该客户

#### 2.2 商品筛选
**用途**：查看某个商品的销售情况

**操作**：
1. 点击"全部商品"下拉框
2. 输入商品名称或编码搜索
3. 选择目标商品
4. 数据自动刷新

**效果**：
- 趋势图显示该商品的销售趋势
- 客户排行显示购买该商品的客户排行
- 所有统计数据仅统计该商品

#### 2.3 组合筛选
**用途**：精准查询特定客户购买特定商品的数据

**示例场景**：
- "张三购买昆仑机油的历史记录"
- "李四购买柴油的消费情况"

**操作**：
1. 选择客户（如：张三）
2. 选择商品（如：昆仑机油）
3. 设置日期范围（或清除日期查看全部）
4. 查看精准统计数据

**效果**：
- 只显示张三购买昆仑机油的数据
- 趋势图、排行榜都受筛选条件影响

#### 2.4 日期筛选
**默认**：最近 30 天

**快捷选项**：
- 最近 7 天
- 最近 30 天
- 本月
- 本年

**自定义**：点击日期选择器，手动选择起止日期

**清除日期** 🆕：
- 点击"清除日期"按钮
- 查看全部历史数据（不限时间范围）
- 适合查看商品/客户的完整销售历史

#### 2.5 重置功能
**用途**：一键恢复默认状态

**操作**：点击"重置"按钮

**效果**：
- 日期范围恢复为"最近 30 天"
- 客户筛选清空
- 商品筛选清空
- 统计粒度保持不变（日/周/月）

### 3. 利润分析（仅管理员）

管理员登录后可查看：
- 利润趋势图
- 商品利润排行
- 客户利润贡献排行
- 总利润、平均利润率、最高利润商品

**支持筛选** 🆕：
- 利润统计同样支持按客户/商品筛选
- 例如：查看"张三"的利润贡献
- 例如：查看"昆仑机油"的利润情况

### 4. 库存统计

**商品类别库存分布**
- 饼图展示各类别库存占比
- 直观了解库存结构

**库存排行 TOP10**
- 显示库存最多的商品
- 便于库存优化决策

**库存不足预警**
- 自动列出库存 ≤ 10 的商品
- 红色标签提醒缺货商品
- 橙色标签提醒库存不足商品

## 📱 移动端使用

### 手机端优化 🆕
- 筛选栏自动纵向排列（不拥挤）
- 图表文字自动放大（清晰易读）
- 按钮加大（易于点击）
- 触摸有反馈（点击缩放效果）

### 操作建议
- **查看统计**：手机完全胜任 ⭐⭐⭐⭐
- **筛选数据**：手机操作流畅 ⭐⭐⭐⭐
- **导出数据**：建议电脑端操作 ⭐⭐⭐

## 💡 使用技巧

### 场景 1：分析客户消费习惯
1. 选择客户（如：张三）
2. 清除日期（查看全部历史）
3. 查看趋势图：了解消费频率
4. 查看商品排行：了解偏好商品

### 场景 2：分析商品销售情况
1. 选择商品（如：昆仑机油）
2. 选择时间范围（如：本月）
3. 查看趋势图：了解销售走势
4. 查看客户排行：了解主要客户

### 场景 3：核对客户账目
1. 选择客户
2. 选择日期范围（如：上个月）
3. 查看总销售额
4. 查看商品明细

### 场景 4：库存管理
1. 查看"库存不足商品预警"
2. 记录需要补货的商品
3. 查看"库存排行"，了解积压商品

### 场景 5：利润分析（管理员）
1. 登录管理员账号
2. 查看利润趋势，了解盈利情况
3. 查看利润率低的商品，考虑调价
4. 查看高利润客户，重点维护

## ⚠️ 注意事项

1. **数据更新**：筛选条件变更后，数据立即刷新
2. **权限限制**：利润相关数据仅管理员可见
3. **性能优化**：客户和商品列表首次加载后会缓存，避免重复请求
4. **历史数据**：清除日期后查询全部数据可能需要 2-3 秒
5. **组合筛选**：客户+商品同时筛选时，结果可能为空（该客户未购买该商品）

## 🔄 常见问题

**Q：为什么选择客户后，客户排行只有一个人？**  
A：这是正常的，因为筛选后只统计该客户的数据。

**Q：清除日期后，图表显示很多数据点，看不清怎么办？**  
A：切换到"按周"或"按月"统计，减少数据点数量。

**Q：手机上图表文字太小怎么办？**  
A：最新版本已优化，图表会自动放大字体。如果还是小，可以横屏查看。

**Q：筛选后没有数据怎么办？**  
A：检查筛选条件是否过于严格，或者该组合确实没有数据。点击"重置"恢复默认。

**Q：利润统计看不到？**  
A：需要用管理员账号登录（用户名：admin，密码：123456）。

## 🎓 进阶用法

### 定期分析报告
**每周一**：
1. 查看上周销售趋势
2. 对比上上周，分析变化
3. 记录 TOP 商品和客户

**每月末**：
1. 查看本月总销售额
2. 分析利润情况（管理员）
3. 检查库存预警
4. 规划下月进货

### 客户维护
**识别 VIP 客户**：
1. 查看客户消费排行
2. 重点维护 TOP10 客户
3. 定期回访、优惠活动

**唤醒沉睡客户**：
1. 清除日期，查看全部客户
2. 对比近期排行，找出流失客户
3. 电话回访，了解原因

### 库存优化
**减少积压**：
1. 查看库存排行
2. 对库存高、销量低的商品促销

**避免断货**：
1. 查看库存不足预警
2. 及时补货热销商品
